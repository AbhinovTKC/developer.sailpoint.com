## Create a new Schema Attribute for Non-Employee Source



This API creates a new schema attribute for Non-Employee Source. The schema technical name must be unique in the source. Attempts to create a schema attribute with an existing name will result in a &quot;400.1.409 Reference conflict&quot; response. At most, 10 custom attributes can be created per schema. Attempts to create more than 10 will result in a &quot;400.1.4 Limit violation&quot; response.&lt;br&gt;&lt;br&gt;
Requires role context of `idn:nesr:create`
